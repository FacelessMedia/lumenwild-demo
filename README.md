# Lumenwild

An original-IP, retro handheld-style creature-taming RPG demo built for the browser.

## Play

```bash
npm install
npm run dev
```

Open the local Vite URL, then use:

- Arrow keys or WASD to move
- Z, Enter, or Space to interact
- X or Escape to open the menu
- On-screen controls on touch devices

## Demo loop

Explore Fernhollow, choose one of three starter wildlings, open the Sunwash Run gate, encounter and befriend wildlings, challenge Scout Niko, and relight Skyglass Beacon. Progress saves to browser local storage.

## Quality checks

```bash
npm test
npm run build
```

## Original-IP statement

Lumenwild uses original names, creatures, dialogue, maps, art, interface styling, progression, and game data. It does not include Pokémon names, artwork, maps, music, story text, source data, or proprietary assets.
